Claude AI + SAP ABAP via MCP

Connect Claude AI directly to your SAP system using MCP (Model Context Protocol) — read, modify, and manage ABAP objects via natural language.

🚀 What is MCP?

MCP (Model Context Protocol) allows AI models to interact with real systems.

  • Read ABAP code
  • Modify objects
  • Create transports
  • Run syntax checks

🏗 Architecture

Claude ↔ MCP Servers ↔ SAP ADT API ↔ SAP System

  • mcp-abap-adt → Read-only
  • mcp-abap-abap-adt-api → Full CRUD

📦 Prerequisites

  • Windows 10/11
  • SAP system with ADT enabled
  • SAP_ADT_DEVELOPER role
  • Claude subscription

⚙️ Installation

1. Install Node.js

node -v
npm -v

2. Install Claude CLI

npm install -g @anthropic-ai/claude-code
claude

3. Clone MCP Server

git clone https://github.com/mario-andreschak/mcp-abap-adt
cd mcp-abap-adt
npm install
npm run build

4. Example .env

SAP_URL=https://your-system
SAP_USERNAME=USER
SAP_PASSWORD=PASS
SAP_CLIENT=300
